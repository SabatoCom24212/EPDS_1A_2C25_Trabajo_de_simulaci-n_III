# Trabajo de simulación III (EPDS)

Este trabajo está hecho con **Python** en un **Jupyter Notebook** ([https://jupyter.org/](https://jupyter.org/)).

Para la manipulación y preparación de los datos de la encuesta salarial, utilizamos las librerías **Pandas** ([https://pandas.pydata.org/](https://pandas.pydata.org/)) y **NumPy** ([https://numpy.org/](https://numpy.org/)).

Para la generación de los gráficos usamos **Matplotlib** ([https://matplotlib.org/](https://matplotlib.org/)) y **Seaborn** ([https://seaborn.pydata.org/](https://seaborn.pydata.org/)).

Para la modelización predictiva y la validación de la importancia de las variables salariales, utilizamos la librería **Scikit-learn** ([https://scikit-learn.org/stable/](https://scikit-learn.org/stable/)).

Finalmente, renderizamos el PDF con **Quarto** ([https://quarto.org/](https://quarto.org/)).
